    PDF.js — Inline PDF viewer

    Mozilla's PDF.js renders a sample document directly onto a canvas inside a maud-ui shell. The maud-ui toolbar drives the imperative API — page navigation, zoom, fit-to-width. The seed PDF is a small in-browser generated file so the demo has zero external dependencies.
